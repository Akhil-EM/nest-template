Rename .env.eg to .env

Remove packages from packages.json according to your database needs

Visit http://localhost:PORT/documentation for swagger documentation.

Run
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
